# Cassino e Lanchonete

Mini-sistema em Python que simula a entrada de clientes em um cassino e a compra de produtos em uma lanchonete.

O sistema faz:
- Validação de nome
- Verificação de idade
- Autorização para menores acompanhados
- Menu de compra
- Cálculo automático do valor total

## Tecnologias e conceitos
- Python
- Funções
- Laços de repetição
- Condições
- Dicionários
- Tratamento de erros

Projeto criado para praticar lógica de programação e construção de sistemas em Python.
