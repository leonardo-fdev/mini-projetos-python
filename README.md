# Mini Projeto em Python

Este repositório reúne pequenos projetos desenvolvidos durante meus estudos em Python e lógica de programação.  
Cada projeto aqui representa um passo da minha evolução prática, focado em transformar ideias em sistemas funcionais.

Os projetos foram criados para praticar:
- Estruturação de programas
- Funções e organização de código
- Lógica de negócio
- Entrada e validação de dados
- Simulação de sistemas reais

## Objetivo
Construir um portfólio prático voltado para desenvolvimento e área de dados, mostrando minha evolução como programador.

## Projetos
- Cassino e Lanchonete  
  Sistema em Python que simula a entrada de clientes em um cassino e a compra de itens em uma lanchonete via terminal.
